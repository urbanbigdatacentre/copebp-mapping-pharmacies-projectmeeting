Initial setup

The workflow uses different libraries such as haven and readxl for data management as well as sf, dplyr and leaflet for geospatial data management.

Finding the average distance

Data inputs are: - the distance matrix (one per city) between the pharmacies. This is expressed in meters. - the clusters defined

The analysis consists on grouping pharmacies (start, end) to their respective clusters and calculate the average distance between pharmacies in the same cluster.
